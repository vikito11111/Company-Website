## ğŸ”§ Customization

### Change Business Information
Edit the contact section in `index.html`:
```html
<p><strong>Address:</strong> 123 Beauty Lane | <strong>Phone:</strong> (555) 123-4567</p>
```

### Add Gallery Images
Replace placeholder gallery items in `index.html`:
```html
<div class="gallery-item">
    <img src="your-image.jpg" alt="Before and After">
</div>
```

### Modify Colors
Update color variables in `styles.css`:
- Change `#d4af37` to your accent color
- Change `#1a1a2e` to your primary dark color

### Update Services
Add or remove service cards in the services section of `index.html`
